# **App Name**: Tadoba Wheels

## Core Features:

- Cab Booking: Allow users to book a cab from Nagpur to Tadoba.
- Roadmap Display: Show a clear roadmap from Nagpur to Tadoba with key landmarks.
- Safari Information: Display information about Tadoba safaris, timings, and booking details.
- Tiger Information: Fetch information about Tadoba tigers using AI.
- AI Content Tool: AI will decide when, or if, to incorporate different information pieces in its output.
- Tariff Page: Display cab tariffs and pricing information.
- Tiger Count Animation: Animate the total tiger count at the bottom of the page.

## Style Guidelines:

- Primary color: Forest Green (#388E3C) to evoke nature and wildlife.
- Background color: Light Green (#E8F5E9), a desaturated version of the primary color.
- Accent color: Yellow-Green (#9ACD32) to highlight key elements and CTAs.
- Body and headline font: 'PT Sans', a humanist sans-serif that is both modern and warm.
- Use nature-inspired icons (e.g., tiger paw, leaves, road signs).
- Clean and modern layout with a focus on readability.
- Smooth transitions and subtle animations for an engaging user experience.
